# Node.js Learning Lab

A collection of Node.js, Express.js, EJS, routing, filesystem, and backend development exercises completed as part of coursework and personal practice.

## Topics Covered

* Node.js Fundamentals
* Express.js
* EJS Templating
* Routing
* Middleware
* File System Module
* Callback Functions
* Email Handling
* Static File Serving

## Purpose

This repository serves as a learning workspace containing multiple small projects and exercises used to understand backend development concepts.

The code in this repository is intended for educational and practice purposes and may contain experimental implementations while learning various Node.js concepts.
